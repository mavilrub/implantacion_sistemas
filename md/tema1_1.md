
# Instalación del sistema operativo

En este apartado se estudian los pasos necesarios para instalar un sistema operativo:

1. Preparación del hardware.
2. Selección del sistema operativo (Windows, Linux, etc.).
3. Tipos de instalación (limpia, actualización, dual boot, red, desatendida).
4. Particionado de discos y sistemas de archivos.
5. Instalación de controladores y comprobación del arranque.

---

**Ejemplo práctico:**  
Instalación de **Ubuntu Server 24.04 LTS** configurando LVM y acceso SSH remoto.
