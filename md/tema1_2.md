# Configuración inicial y herramientas del sistema

Tras instalar el sistema operativo, es necesario realizar una configuración básica:

- Ajuste de fecha, hora y zona horaria.  
- Creación de usuarios y grupos.  
- Configuración de red y hostname.  
- Instalación de actualizaciones.  
- Comprobación de servicios del sistema.

---

**Herramientas comunes:**

- `ifconfig` / `ip addr` → Ver configuración de red  
- `hostnamectl` → Cambiar nombre del equipo  
- `systemctl` → Gestionar servicios  
- `top` / `htop` → Supervisar procesos
